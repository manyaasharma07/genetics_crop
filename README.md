# Genetic Traits & Crop Recommendation System

This repository contains the front-end for the Genetic Traits and Crop Recommendation System developed for RV College of Engineering.

## Development

Prerequisites: Node.js and npm/yarn.

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- Tailwind CSS

## Notes

Replace assets or logo in the `public/` folder as needed. The project entry is `src/main.tsx` and the landing page is `src/pages/Index.tsx`.
